## Getting Started

Its very easy to get started with multiselct

Only three steps invloved..

1.  ```js
    npm install @ngx-lib/multiselect --save
    ```

2.  ```js
    import { NgxMultiselectModule } from '@ngx-lib/multiselect';
    ```

3.  ```html
    <ngx-multiselect 
        [(ngModel)]="selectedOption" 
        [options]="options" 
        ngDefaultControl>
    </ngx-multiselect>
    ```
