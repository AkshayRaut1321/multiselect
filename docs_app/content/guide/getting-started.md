# Getting Started

Its very easy to get started with multiselct

1.  `npm install @ngx-lib/multiselect --save`

2.  `import { NgxMultiselectModule } from '@ngx-lib/multiselect';`

3.  ```html
    <ngx-multiselect 
        [(ngModel)]="selectedOption" 
        [options]="options" 
        ngDefaultControl>
    </ngx-multiselect>
    ```
